Wizard Through Layout

Customer Profile:

Wizard element injections are based on field to template region bindings which are defined by the templates properties of the field view parameter. Wizard template uses alpaca-wizard-step class to mark the wizard step. This example loades template, data, schema and options parameters through ajax calls.
